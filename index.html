📥 How to import into Mailjet

  1. Download each HTML file above
  2. Go to Mailjet → Content → Email Templates
  3. Click "Create a template"
  4. Choose "By coding it in HTML"
  5. Choose "Import HTML from a file"
  6. Upload the downloaded file & name it after the event
  7. Save — your template is ready! 🎉
